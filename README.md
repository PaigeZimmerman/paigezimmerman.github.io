### 3D Virtual Tour of Va Pensiero, Historic Equestrian Facility in Hunterdon County, NJ

Below is a 3D Google Street View Tour of the equestrian facility that I work at! The first image was taken from the path that goes from the main amenities towards the pastures. The second image in the tour was taken from behind the barn. This facility is very large, almost 200 acres of land, and it is historic/ famous in the area of Hunterdon County, New Jersey. The barn used to be home to George Morris, legendary jumping rider, among multiple other famous equestrian professionals, including Olympic rider Robert Dover. That was over 30 years ago, and the facility now has a new owner, so it has had a lot of updates over the years. I feel very lucky to be able to work out of such a beautiful facility!

### The 2022 Election: Where are People Tweeting About it?

Below is a map that shows where people are tweeting about the upcoming election. I collected data from Twitter using a data collection site called Social Bearing. I collected recent data about where across the country people are posting the hashtag #2022election. I then cleaned that data and created an excel file of it, which I used in combination with ArcGIS to create this map. If we zoom in, we can see that people are posting more about the 2022 election around large cities, such as Denver, Los Angeles, Chicago, Philidelphia, and more.
